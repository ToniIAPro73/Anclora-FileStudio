# Desktop PRO UX Decisions

## Product Positioning

Desktop PRO is the complete local edition:

```text
Web = rápido y sin instalar.
Desktop PRO = Web + motores nativos + historial + diagnóstico + lotes pesados.
```

## Navigation

Use task-oriented tabs instead of a single generic converter:

- Imágenes
- PDF
- Audio y vídeo
- Documentos
- OCR
- Archivos
- Ebooks
- Más herramientas
- Historial
- Diagnóstico

## Labels

All inputs/selects must have visible labels. Required fields must be clear.

## Copy

Keep first-level copy non-technical. Engine names can appear in details,
diagnostics and capability badges.

## Web Reuse

Desktop can reuse Web tools for quick local processing. Native PRO sections
must clearly state which extra formats and larger workloads require local
engines.
